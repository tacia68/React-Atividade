#Loja Virtual - Projeto WebAcademy
Este é um projeto desenvolvido durante a disciplina do curso WebAcademy, e consiste em uma aplicação de Loja Virtual
composta por dois sistemas: backend e frontend. O objetivo da aplicação é permitir o cadastro,
edição e remoção de produtos, além de oferecer a funcionalidade de adicionar produtos ao carrinho de compras.

#Funcionalidades
Cadastro de produtos: Permite aos usuários cadastrar novos produtos na loja.
Edição de produtos: Permite a edição dos dados de produtos existentes.
Remoção de produtos: Possibilita a remoção de produtos da loja.
Adicionar ao carrinho: Os usuários podem adicionar produtos ao carrinho de compras para futura compra.

#Execução do Projeto

No repositorio: https://github.com/tacia68/React-Atividade
devererá acessar a pasta
cd LojaVirtual

..

cd backend
docker compose up

..
abrir um novo terminal
cd backend
npm install
npm start

abrir um novo terminal
cd frontend
npm install
npm start


---------
depois da execução do frontend poderá acessar o endereço:
